# remote-work-effect
# Group Data Analysis Project Plan  
**Topic:** Impact of Remote Work on Productivity and Employee Morale  
**Final Output:** One well-structured, insight-driven dashboard

---

## 1. Project Objective
Analyze how **remote, hybrid, and on-site work** affect:
- **Employee productivity**
- **Employee morale**

Then propose an **ideal post-pandemic work policy** based on data-driven insights.

---

## 2. Key Principle
> **Different responsibilities, one story, one dashboard**

Only **one final dashboard** is submitted.

---

## 3. Datasets Overview
- **Dataset A:** Productivity & work setup metrics  
- **Dataset B:** Employee morale & well-being metrics  

The datasets are integrated using a **shared linking field** (e.g., employee_id, department, work_mode).
